# Frontend

Next.js web dashboard for SSL certificate monitoring.

## Structure

```
frontend/
├── src/
│   ├── app/
│   │   ├── globals.css          # Tailwind CSS styles
│   │   ├── layout.js            # Root layout
│   │   └── page.js              # Home page
│   └── components/
│       └── CertificateChecker.jsx   # Main component
│
├── out/                         # Static export (generated by build)
├── next.config.js               # Next.js configuration
├── package.json
└── tailwind.config.js
```

## Configuration

Create `.env.local` to configure domains:

```bash
NEXT_PUBLIC_DOMAINS_TO_CHECK=yourdomain.com;anotherdomain.com;thirddomain.com
```

Domains are semicolon-separated. Defaults to `google.com;microsoft.com`.

## Development

```bash
npm install
npm run dev
```

Runs on `http://localhost:5000`. Proxies `/api/*` requests to `http://localhost:3000` (backend).

**Note:** Backend must be running for the dashboard to work.

## Production Build

```bash
npm run build
```

Creates static files in `out/` directory, served by the backend in Docker.

## Features

- Real-time certificate status checking
- Color-coded status indicators:
  - Green: Valid (30+ days)
  - Yellow: Expiring soon (< 30 days)
  - Red: Expired
- Responsive design with Tailwind CSS
